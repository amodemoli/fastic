#### ▲ STEP4 • USE DEFAULT SERVER MIDDLEWARES

❝ **How Can I Use Middlewares In All Requests?**
you can add middlewares to all your server requests, for middleware on custom server requests please goto "**[STEP5](https://github.com/amodemoli/fastic/blob/main/docs/s5.md)**", fastic web framework has default middlewares for all requests you can use, for example: `RequestLogger`,`CORS`,`Security Headers`, `Request Limiter`, `Panic Recover`. you can use this. `CORS` && `Security Headers` middlewares have settings on env file. you can create env file with `app.Env.Create()` method, read the documention for env files on "**[STEP2](https://github.com/amodemoli/fastic/blob/main/docs/s2.md)**".

```go
// 🗁 main.go file contents
package basic

import (
	"github.com/amodemoli/fastic/core/fastic"
	"github.com/amodemoli/fastic/core/middleware"
)

func main() {
	app := fastic.New()

	app.GET("/ping", func(c *fastic.Ctx) {
		c.RawJSON(`{"message": "pong"}`)
	})

	// use SChain method for create new chain for all requests.
	handler := app.Mdw.SChain( // create new var named handler
		app.Handler, // first you need write the server handler to chain.
		middleware.Recovery, // added recovery middleware (default middlewares are on middleware package of fastic)
		// if are you using the Recovery middleware please write thi middleware first.
		// next you can add RequetsLogger (this middleware is not good for real servers because it slows down the server speed)
		middleware.CORS(app), // some middleware's need to app please put your app on middlewars.
		// if are you added CORS you can edit CORS setting's on ".env" file.
	)
	// you can create new middlewares and add this middlewares on there, your middleware will run in all request responses,
	// please dont use this. read "STEP5" and learn how you can create custom middleware's for custom requets.
	// app.Mdv.Chain middlewares can slow your server. WARNING!!

	app.Run(handler) // put new handler on app.Run.
}
```

please dont add many middlewares on `app.Mdv.Chain`.
you can add custom middlewares for custom paths you can learn this on next step.
middlewares: `Recovery` && `CORS` && `Security Headers` they are need to add on `app.Mdw.SChain`, they can help you on real projects but i do not suggest you adding `Request Logger` middleware on your server. because this middleware can up to 30% slow your server.
