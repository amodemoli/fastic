#### ▲ STEP1 • INSTALL AND RUN FIRST APP

❝ **How To Install Fastic Framework?**
for install framework you need install golang on your system.
use this commands for create new project of golang:

```bash
go version # for check your golang
go mod init <name> # write your project name
touch main.go # use ni main.go for power shell
```

next you need to install fastic framework:

```bash
go get github.com/amodemoli/fastic@latest
```

done. fastic installed on your project. lest coding...

❝ **Writing First Webserver With Fastic**
okey, now open your editor and write this code on `main.go`:

```go
// 🗁 main.go file contents
package basic

import "github.com/amodemoli/fastic/core/fastic"

func main() {
    // create new application with fastic.New() method.
	app := fastic.New()

    // use "GET" method, with "/ping" path, i create new ctx function and used c.WriteString for write string on web server.
    // c is your context on router.
	app.GET("/ping", func(c *fastic.Ctx) {
		c.WriteString("pong!")
	})

    // now i use app.Run function for run web server and for handler i use app.Handler value.
    // default server port is 8000 you can edit port on ".env" file. (.env creating and using is for step2)
	app.Run(app.Handler)
}
```

now your server is runned on port 8000, if you need custom port or edit your website settings you need to edit values on ".env" file or more! in "**[STEP2](https://github.com/amodemoli/fastic/blob/main/docs/s2.md)**" i learn you how you can create env file with custom values. =D
now run this command for run the server:

```bash
go run . # or you can use "go run main.go"
```
now visit the http://localhost:8000/ping. you see **"pong"** message =D
