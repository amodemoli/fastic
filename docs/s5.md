❝ **How Can I Write A New Middleware?**
i learn to you, how you can create one middleware =D this is very easy,
_you can ask to ai for middleware on web services._

create my first middleware:

```go
// 🗁 login.mdw.go file contents
package basic

import (
	"github.com/amodemoli/fastic/core/fastic"
)

func exampleAuth(next fastic.RequestHandler) fastic.RequestHandler {
	return func(c *fastic.Ctx) {
		// write your codes
		// i need to write new auth (fake auth)
		loggedIn := false
		if !loggedIn {
			c.SetStatusCode(401) // you can change the page status code
			c.RawJSON(`{"message: "you need login first"}`)
			return // exit from request and dont run user request
		}

		// if user logged, they call the next function (target request).
		next(c)
	}
}
```

❝ **Now How Can I Use Middleware??**
you can add your middleware or middlewares on one path:

```go
// 🗁 main.go file contents
package basic

import (
	"github.com/amodemoli/fastic/core/fastic"
	"github.com/amodemoli/fastic/core/middleware"
)

func main() {
	app := fastic.New()

	app.GET("/dashboard", func(c *fastic.Ctx) {
		c.RawJSON(`{"message": "welcome to dashboard"}`)
	}, exampleAuth) // add your middleware here you can add ... middleware's on there.

	app.Run(app.Handler)
}
```

okey now run the server:

```bash
go run . # if your middleware writed on another file please dont use "go run main.go"
```

now you can check the http://localhost:8000/dashboard you can see the error, if you change `loggedIn := false` you can connect to the dashboard.

❝ **Example Of Adding +2 Middleware On Path:**
creating new middleware:

```go
// 🗁 admin.mdw.go file contents
package basic

import (
	"github.com/amodemoli/fastic/core/fastic"
)

func exampleAdminAuth(next fastic.RequestHandler) fastic.RequestHandler {
	return func(c *fastic.Ctx) {
		// write your codes
		// i need to write new admin auth (fake auth)
		isAdmin := false
		if !isAdmin {
			c.SetStatusCode(403) // you can change the page status code
			c.RawJSON(`{"message: "only admins can enter on this page"}`)
			return // exit from request and dont run user request
		}

		// if user logged, they call the next function (target request).
		next(c)
	}
}
```

now i adding two middlewares on `/admin` path, server first check the user login status and next server check's user is admin or no.

```go
// 🗁 main.go file contents
package basic

import (
	"github.com/amodemoli/fastic/core/fastic"
	"github.com/amodemoli/fastic/core/middleware"
)

func main() {
	app := fastic.New()

	app.GET("/admin", func(c *fastic.Ctx) {
		c.RawJSON(`{"message": "welcome to dashboard"}`)
	}, exampleAuth, exampleAdminAuth) // first check user login next check the admin.
    // you can add ... middlewares. =D

	app.Run(app.Handler)
}
```
