❝ **How Can I Send RestApi Data?**
you can send json response with `c.RawJSON` or `c.JSON`, or read the response with `c.Bind`. see example of send json:

```go
// 🗁 main.go file contents
package main

import "github.com/amodemoli/fastic/core/fastic"

func main() {
	app := fastic.New()

	// create new addres for your api
	app.GET("/api/info", func(c *fastic.Ctx) {
		c.RawJSON(`{"name": "amir"}`) // response with "RawJSON" or "JSON" to frontend.
		// frontend can read this value. =D
	})

	app.Run(app.Handler)
}
```

❝ **How Can I Read RestApi Data?**
you can use `c.Bind` method for read the json data =D
first you need to create new struct for save json data =

```go
// 🗁 main.go file contents
package main

import (
	"fmt"

	"github.com/amodemoli/fastic/core/fastic"
)

// create new struct for save json data
type Response struct {
	Name string `json:"name"` // on json this field saves to Name = "name", on : `json:"name"`
}

func main() {
	app := fastic.New()

	// use post method, because i need to get response. =D
	app.POST("/api/info", func(c *fastic.Ctx) {
		var user Response // create new var with type Response

		if err := c.Bind(&user); err != nil {
			c.Status(400).RawJSON(fmt.Sprintf(`{"error": "%v"}`, err))
			return
		}

		// show the name =D
		c.RawJSON(fmt.Sprintf(`{"name": "%s"}`, user.Name))
	})

	app.Run(app.Handler)
}
```

you can use this method and read the json data =D
