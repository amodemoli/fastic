#### ▲ STEP3 • WRITE RESPONSE AS JSON TO USER

❝ **How Can I Write JSON Response To User?**
on fastic for write json response you have two methods, `app.JSON` and `app.RawJSON`, lets try methods:

```go
// 🗁 main.go file contents
package basic

import (
	"github.com/amodemoli/fastic/core/fastic"
)

func main() {
	app := fastic.New()

	app.GET("/ping", func(c *fastic.Ctx) {
		c.JSON(map[string]string{ // or map[any]any
			"message": "pong",
		})
	})

	app.Run(app.Handler)
}
```

you can see `c.JSON` method, for use this method you need to create map.
lets go to see `c.RawJSON` method:

```go
package basic

import (
	"github.com/amodemoli/fastic/core/fastic"
)

func main() {
	app := fastic.New()

	app.GET("/ping", func(c *fastic.Ctx) {
		c.RawJSON(`{"message": "pong"}`) // dont use maps
	})

	app.Run(app.Handler)
}
```

`c.RawJSON` method is easy and speed. you can use this method for small json responses. and can use `fmt.SprintF` for drop valuse on json.
run server and see:

```bash
go run . # or "go run main.go"
```

u can visit the http://localhost:8000/ping url after running server.
