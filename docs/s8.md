#### ▲ STEP8 • ATTACHMENT FILES TO USER BROWSER FOR DOWNLOAD

❝ **How Can I Send Custom File To Download?**
thats easy, you can use `c.Attachment` method for this =D see example:

```go
// 🗁 main.go file contents
package main

import (
	"github.com/amodemoli/fastic/core/fastic"
)

func main() {
	app := fastic.New()

	app.GET("/download", func(c *fastic.Ctx) {
		// you need to add your target file path and file name
		if err := c.Attachment("$_PATH/myfile.go"); err != nil { // add your filepath for download =D
            // they cannot find or read stats of your target file. handle error =D
        }
	})

	app.Run(app.Handler)
}
```

and done =D run the server with:

```bash
go run . # or "go run main.go"
```

and go to http://localhost:8000/download , your browser started to downloading file =D
