#### ▲ STEP6 • SERV FRONTEND FILES (HTML,CSS,JS)

❝ **How Can I Serv My FrontEnd Files?**
you can serv your `.html`,`.css`,`.s` files, with `c.SendFile` method.
on `c.SendFile` method you can write your custom `.html` file path and link your css and js file to file. see example:

```go
// 🗁 main.go file contents
package basic

import (
	"github.com/amodemoli/fastic/core/fastic"
)

func main() {
	app := fastic.New()

	app.GET("/", func(c *fastic.Ctx) {
        // write this method and write target file path.
		c.SendFile("$_PATH/public/index.html")
	})

	app.Run(app.Handler)
}
```

for connect front end and backend you need to use `ResApi` or `gRPC`, read next step for learn `RestApi`. =D
