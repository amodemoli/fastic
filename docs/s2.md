#### ▲ STEP2 • ENV FILE AND SERVER SETTINGS

❝ **How To Create Env File?**
you can edit you server port, development port status, request cors and security headers. you need create ".env" file (only for development server for real server please dont use .env file use values example: export VARNAME=value).
you can create new env file with `app.Env.Create()` method.

```go
// 🗁 main.go file contents
package basic

import "github.com/amodemoli/fastic/core/fastic"

func main() {
	app := fastic.New()

    // this method create new ".env" file with default value's on this path,
    // app.Env.Create return's a boolian var. you can see env file load stats
	if !app.Env.Create() {
        // app print method is a method for print values on terminal with custom pattern
        // this function just for clean ui, if you need to print some string on terminal please use app.Print or app.PrintB (print B get byte for performance and speed)
		app.Print(color.Red , "ERROR", `cannot load ".env" file.`)
	}

	app.Run(app.Handler)
}

```

this method make's new `.env` file on you correct path (the function caller path).
run server with this command for create ``.env`` file.
```bash
go run . # or "go run main.go"
```
you need to add `CORS` && `SECURITY HEADERS`, if you dont add this middlewares on default requests. security header and cors settings are not working on `.env` file.
